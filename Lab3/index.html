Помощь

Присоедениться к каналу: /join имя_канала
Создать частный (приватный) канал: /join
Пригласить кого-либо (например в частный канал): /invite имя_пользователя
Отозвать приглашение: /uninvite имя_пользователя
Выйти из чата: /quit
Приватное сообщение: /msg Имя_пользователся Текст
Открыть приватный канал: /query Имя_пользователя
Закрыть приватный канал: /query
Описать действия: /action Текст
Описать действия в приватном сообщении: /describe Имя_пользователя Текст
Игнорировать/принимать сообщения от пользователей: /ignore Имя_пользователся
List ignored users: /ignore
Display user channel: /whereis Username
Выгнать пользователся (Kick) (только модераторы и администраторы): /kick Имя_пользователя [Минуты]
Разбанить пользователя (Только модераторы и администраторы): /unban Имя_пользователя
Список забаненных пользователей (Только для модераторов): /bans
Показать IP пользователся (Только модераторы и администраторы): /whois Имя_пользователя
Список доступных пользователей: /who [Имя_канала]
Список доступных каналов: /list
Бросить кубики: /roll [количесво]d[количество сторон]
Change username: /nick Username
